01

Can a waiter build an order before the table is ready to send?

Yes. The waiter builds the cart for any table and it saves automatically to local storage. The cart survives page refreshes and is specific to that table. When the customer is ready, the waiter sends the whole cart to the KDS in one tap.

02

Can different customers at a table send their orders separately?

Yes. Each person confirms their items and the waiter queues them individually. When everyone is ready, the waiter sends the full cart or sends per-person if preferred and add the other items later.

03

How do menu modifiers work on the waiter app?

When adding an item, a modifier panel opens automatically. The waiter selects the options, adds any notes, and the customized item goes into the table cart with all modifiers attached they can select from commo modifiers or ones specific to menu item.

04

How does split billing work from the waiter side?

The waiter taps the split bill button, selects either to do it them self and select the split type (equal, by item or by amounts), and sends a pre-filled split request directly to the POS cashier Then cashier sees the split immediately and processes each payment or they can select the other option to let the cashier handle it and just select which type the customers want. The cashier sees the split immediately and processes each payment.

05

Can the waiter filter tables by their own assignment?

Yes. The floor view has filter tabs My Tables, All Tables, and by status (Ordering, Billing, Served etc). Waiters only see what is relevant to them.

06

Does the waiter app work offline?

Cart data is saved to local storage so it persists on the device. The orders can fire using LAN mode even without any data as long as the device is on the same WIFI as the KDS or Once connectivity is restored, the order syncs and fires to the KDS automatically.

07

Can the waiter edit the cart after items are added?

Yes. Items can be added, removed, or quantity-adjusted any time before the cart is sent. Notes and modifiers can also be edited before sending.

08

Does the KDS receive modifier details?

Yes. Every modifier, note, and special request appears on the KDS ticket so the kitchen has full context without any verbal relay.

09

Do You sale hardware and can your software run our current ones?

Yes we do sell hardware, reach out to get a quote and its not a mast to use our hardware our software's all works on any device even your existing ones
