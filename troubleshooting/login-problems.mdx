Login problems typically stem from wrong credentials, expired sessions, or network issues. This page walks through resolving each.

## Overview

Common symptoms:

- "Invalid PIN"
- "Session expired"
- Login screen doesn't load
- Wrong branch after login
- 2FA code rejected

## Common mistakes

<Warning>
  Do not repeatedly try wrong PINs. After 5 failures the account is locked for 15 minutes.
</Warning>
