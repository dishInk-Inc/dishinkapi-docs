Network printers connect over Ethernet or WiFi and can serve multiple terminals — ideal for busy venues with several POS and KDS stations.

## Overview

A network printer has its own IP address on the local network. Multiple terminals send print jobs to the same IP.

</Frame>

<Note>
  Static IPs are essential. Dynamic IPs cause silent print failures when the router reboots.
</Note>

## Tips & best practices

<Tip>
  Label each printer physically with its IP and role ("Kitchen — 192.168.1.50"). Saves troubleshooting time.
</Tip>

## Common mistakes

<Warning>
  Do not put network printers on guest WiFi. Use a dedicated staff network for reliability.
</Warning>
