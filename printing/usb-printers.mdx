USB printers are the simplest to set up — plug in, install once, print forever. Best for a single POS terminal that owns its own receipt printer.

## Overview

Supported printers: any 58 mm or 80 mm ESC/POS-compatible thermal printer with a USB interface.

## Set up a USB printer

<Frame caption="USB printer setup">
  ![USB printer](/images/printing/usb.png)
</Frame>

<Note>
  USB printers only work on the terminal they are plugged into. For shared kitchen printing, use network printers.
</Note>

## Tips & best practices

<Tip>
  Buy the exact same printer model for every terminal. Consistent hardware means one set of drivers and one troubleshooting flow.
</Tip>

## Common mistakes

<Warning>
  Do not unplug a printer during service. Windows and Android sometimes lose the driver connection until reboot.
</Warning>
