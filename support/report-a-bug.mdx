Bugs happen — we take every report seriously. This page shows how to file one for the fastest fix.

## Overview

A great bug report contains:

- What happened (observed)
- What you expected
- Exact steps to reproduce
- Terminal, OS, browser (if applicable)
- Screenshot or short screen recording
- Rough transaction reference or time
